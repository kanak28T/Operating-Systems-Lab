# Operating-Systems-Lab

📌 Experiments List


🔹 Experiment 1: Basic Linux Commands

Implementation and demonstration of commonly used Linux commands on Ubuntu such as:

ls, cd, pwd
mkdir, rmdir
cp, mv, rm
cat, echo, man


🔹 Experiment 2: Process Management Commands

Study and usage of process-related Linux commands:

ps
top
kill
nice
bg, fg


🔹 Experiment 3: Process Creation (Parent & Child)

C program demonstrating process creation using:

fork()
Parent and child process identification
Process IDs and execution flow


🔹 Experiment 4: FCFS CPU Scheduling

C program to implement:

First Come First Serve (FCFS) scheduling algorithm
Calculation of:
Waiting Time
Turnaround Time


🔹 Experiment 5: SJF CPU Scheduling

C program to implement:

Shortest Job First (SJF) scheduling algorithm
Non-preemptive scheduling
Waiting and turnaround time calculation


🔹 Experiment 6: Priority CPU Scheduling

C program to implement:

Priority-based CPU scheduling
Execution based on process priority
Performance metrics calculation


🔹 Experiment 7: Banker’s Algorithm

Implementation of Banker’s Algorithm to:

Avoid deadlock
Determine safe sequence
Check system safety state


🔹 Experiment 8: FIFO Page Replacement

C program to implement:

First In First Out (FIFO) page replacement algorithm
Page fault calculation


🔹 Experiment 9: Basic Shell Commands & Scripts

Introduction to shell scripting:

Writing basic shell scripts
Executing commands through scripts
File handling via scripts


🔹 Experiment 10: If-Else in Shell Script

Shell script demonstrating:

Conditional statements (if, else, elif)
Decision-making logic in bash
